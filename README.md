# almost-quantify-web

# 命令

- 安装
    > npm install
- 运行
    > npm run dev
- 编译
    > npm run build

- build for production and view the bundle analyzer report
    > npm run build --report

## 目录结构
- build: 编译脚本
- config: 项目配置，包括dev和发布
- dist: 目标文件
- src: 源文件
  - api: 接口
  - assets: 资源，图片，图标等
  - components: 组件
  - router: 路由表
  - store: store层，接口商店
  - styles: 样式
  - utils: 工具
  - views: 页面

  - main.js: 入口
